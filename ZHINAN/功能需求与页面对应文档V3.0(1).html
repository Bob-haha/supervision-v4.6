二、新建任务页 /task/new

编号功能名称描述涉及数据表操作
N-01~N-10基础信息/类型/标签/主办协办/流程模板/明细/子任务预分配/来源/紧急程度创建任务完整录入tasks,task_types,flow_templates,departments等写入
N-11 新增任务归档任务完成后创建人可归档（详情页或批量归档），status='archived'，不再显示进行中列表tasks写入

三、任务详情页 /task/:id

六大区域：任务头栏、责任矩阵、环节流程、子任务、动态与批示、数据明细区

关键调整功能

编号功能名称调整后描述涉及数据表操作
D-05关注星标当前用户peerId加入/移出 tasks.starredBy 数组，星标高亮tasks(starredBy)读写
D-26动态时间线基于activityLog数组渲染，类型含stage_progress, feedback, leader_comment, urge, external_notification等tasks(activityLog)读取
D-28领导批示输入关领导专用框提交，向 activityLog 追加 type='leader_comment'，P2P通知授权人员tasks(activityLog)写入
D-30负责人确认反馈确认后更新 confirmedFeedback 字段，并追加确认记录到 activityLogtasks(confirmedFeedback, activityLog)读写
D-40 新增动态记录管理所有操作统一写入 activityLog 数组（环节流转、反馈、批示、催办、附件上传、外部通知等），前端动态时间线统一按此渲染tasks(activityLog)写入

其他功能：责任矩阵分派、环节选项、子任务关联聚合、数据明细动态表单等均保留，其中数据明细区行级权限仅创建人可编辑/删除。

提取配置面板(E-01~E-06) 内嵌于详情页，支持执行人指定、URL映射、定时提取及状态查看。

四、流程配置页 /config/flow

编号功能名称调整/描述数据表
F-06制度文件引用 已调整从文件库选policy文件，展示文号、生效日期、失效日期，过期文件红标警告；允许引用但弱提醒；支持选择具体条款(clauses)flow_templates, files
F-01~F-10流程模板新建/环节/选项/约束/从任务保存模板保持原有功能，支持拖拽排序、环节约束、模板文件引用flow_templates, files

五、明细模板配置

模板列表、新建、从字段库勾选、排序删除等 (DS-01~DS-05)

六、字段管理 /config/fields

系统字段增删改，可聚合标记及建议级别 (SF-01~SF-06)

七、类型与标签配置

一级/二级分类、默认模板绑定、标签组管理、Excel导入导出 (T-01~T-07)

八、组织架构管理

部门树维护、人员增删改、批量导入导出 (O-01~O-11)

九、文件管理页 /config/files

编号功能名称调整描述数据表
FL-10文件有效期校验 已调整制度文件列表标红过期；流程配置FilePicker提供“仅显示有效期内”开关，引用时高亮警告；Word上传建议转PDF以保证条款页码准确files(effectiveDate, expiryDate)
FL-01~FL-09分类浏览/上传/版本/统一替换/条款编辑同原始需求，支持版本管理及全文替换files, tasks

十、态势配置页 /config/metrics

编号功能名称调整/细化描述数据表
M-02新建指标 已调整输入名称、级别、可见角色，筛选模式二选一：1)按条件筛选(类型/标签/时间范围至少一个)；2)指定具体任务(多选任务)。聚合字段从system_fields(isAggregatable=true)或特殊值'任务计数(__task_count__)'选取，聚合函数根据字段类型动态SUM/COUNT等。配置保存至metric_definitions.configmetric_definitions, system_fields, tasks
M-01,M-03~M-06指标列表/编辑/删除/排序/推荐标记支持系统内置指标不可删除，推荐指标默认选中metric_definitions

十一、督办看板 /inspector

编号功能名称调整描述数据表
I-06催办操作 已调整快速留言：追加activityLog type='urge'记录并P2P通知；创建督办任务：跳转新建任务并建立task_relations关联tasks(activityLog), task_relations
I-01~I-05,I-07多维筛选/统计图表/导出/黄红灯预警保留原有统计汇总和预警功能tasks

十二、浏览器扩展功能

手动提取邮件/公文生成任务、自动监控表单状态、P2P同步、DOM选择器、一键登录及提取状态上报 (EX-01~EX-07)

十三、权限控制功能

任务删除仅创建人、数据行仅创建人可编、P2P定向同步、督办员全关访问、管理员配置权限、态势数据受P2P范围限制 (P-01~P-07)
